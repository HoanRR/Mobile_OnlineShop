PATCH /api/cart/items/:cart_detail_id Cập nhật số lượng item trong giỏ Auth ✎ Bỏ quantity=0 ▼
Sửa: Bỏ hành vi ẩn quantity=0 = xóa. Muốn xóa dùng endpoint DELETE riêng bên dưới.
Request Body
{ "quantity": 2 // phải >= 1; nếu muốn xóa dùng DELETE }
Responses
200→Cập nhật thành công
400→quantity < 1 hoặc vượt quá tồn kho
404→cart_detail_id không tồn tại

404→cart_detail_id không tồn tại

Responses

POST /api/auth/logout Đăng xuất (vô hiệu hóa refresh token) Auth ▼
Mô tả
Vô hiệu hóa refresh_token trên server (blacklist hoặc xóa). Response 200: { "message": "Đăng xuất thành công" }
